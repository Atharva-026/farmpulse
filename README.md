# FarmPulse

End-to-end AI platform for Indian farmers.

## Features
- Crop recommendation using ML
- Disease detection using CNN
- Loan gateway
- Smart market selling

## Stack
- Frontend: React.js (Vite)
- Backend: Node.js + Express
- Database: MongoDB
- ML Service: Python Flask

## Setup

### Backend
cd backend
npm install
node server.js

### ML Service
cd ml-service
pip install flask scikit-learn pandas numpy flask-cors
python app.py

### Frontend
cd frontend
npm install
npm run dev
